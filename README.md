# EZPassRI


## Description


EZPassRI is a small React application built to help users track and manage toll expenses and receipts in a clean and simple UI. It provides an intuitive expense form to add toll transactions, displays a running list of expenses with totals, and generates printable receipts for each transaction. Designed for quick setup and light customization, EZPassRI is perfect for individuals, small businesses, or transportation managers who want an easy way to log toll costs and keep records by date, vehicle, and amount. The app focuses on usability—offering instant feedback, form validation, and a minimal design—so users can log expenses quickly while keeping a tidy transaction history.


## Features


- Add, edit, and remove toll expense records.
- View a list of recorded expenses and the total amount.
- Generate a simple receipt for each expense.
- Responsive layout suitable for desktop and mobile.
